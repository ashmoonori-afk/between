Usage: between [options] [command]

A local terminal broker for AI pair development.

Options:
  -V, --version     output the version number
  -h, --help        display help for command

Commands:
  init [options]    Create .between/ scaffolding, config, and initial state in
                    the current repo
  status [options]  Print the current phase, cycle, waiting actor, diff hash,
                    and latest event
  start [options]   Start the broker watcher loop (headless file-signal mode)
  pause             Pause the loop
  resume            Resume the loop
  review-now        Force a review of the current diff (unless already reviewed)
  stop              Ask the running broker to stop
  goal <text...>    Lock a new goal for the developer
  approve <scope>   Approve a human-gated action: merge | deploy | promote_rule
  ack               (reviewer helper) acknowledge the outstanding review signal
                    for the current cycle
  doctor            Diagnose the environment and repo for Between
  summarize         Summarize cycle/phase analytics from events.jsonl
  dash [options]    Live broker dashboard (cmux/Kiro-inspired TUI)
  help [command]    display help for command
